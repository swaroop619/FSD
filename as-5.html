What are localStorage and sessionStorage?

localStorage and sessionStorage are part of the Web Storage API in HTML5. They allow websites to store data in the browser.

localStorage stores data with no expiration time — the data persists even after the browser is closed. It is often used to remember user preferences or settings across sessions.

sessionStorage stores data for only one session — the data is cleared when the browser tab is closed. It is useful for temporarily saving form data or user activity during a single session.
